## ADDED Requirements

### Requirement: ヘッダーの装飾アニメーションアイコン

システムはグローバルヘッダーのテーマ切替アイコン（`ThemeToggle`）の隣に、Lottie による装飾用アニメーションアイコンを表示 SHALL する。当該アイコンは装飾目的に限定し、クリック等のインタラクションやページ機能を持たず、既存のヘッダー要素・テーマ切替の挙動を変更しない。

#### Scenario: ヘッダーにアニメーションアイコンが表示される

- **WHEN** いずれかのページのヘッダーが表示される
- **THEN** テーマ切替アイコンの隣に Lottie アニメーションアイコンが表示される
- **AND** 当該アイコンは装飾要素として支援技術に公開されない（`aria-hidden`）

#### Scenario: 常時ループ再生する

- **WHEN** アニメーションアイコンが表示されている
- **THEN** Lottie アニメーションは自動再生で、停止せず継続的にループ再生される

#### Scenario: テーマ切替アイコンとバランスする大きさで表示する

- **WHEN** アニメーションアイコンが表示されている
- **THEN** その大きさはテーマ切替アイコン（`ThemeToggle`）と並んで視覚的にバランスする範囲に収まる

#### Scenario: PC・モバイル双方で表示する

- **WHEN** モバイル幅・デスクトップ幅のいずれでヘッダーが表示される
- **THEN** アニメーションアイコンはどちらの幅でも表示される

#### Scenario: モーション低減設定を尊重する

- **WHEN** 利用者の環境が `prefers-reduced-motion: reduce` を要求している
- **THEN** アニメーションは自動再生されず、静止状態で表示される

#### Scenario: 既存ヘッダー機能に影響しない

- **WHEN** アニメーションアイコンが追加された状態でヘッダーが表示される
- **THEN** ロゴ・ナビゲーションリンク・テーマ切替・モバイルメニューは従来どおり動作する
